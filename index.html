One-time setup — connect your Supabase database
1
Go to supabase.com → New project → choose a name and region → wait ~2 min for it to provision.
2
In your project go to SQL Editor and run this query to create the two tables:

CREATE TABLE cohorts (id text PRIMARY KEY, name text, thresh numeric, students jsonb);
CREATE TABLE student_state (key text PRIMARY KEY, state jsonb);

Then go to Authentication → Policies and enable Row Level Security OFF for both tables (or add an anon policy).
3
Go to Project Settings → API. Copy your Project URL and anon public key and paste them below.
Found in Project Settings → API → Project URL
Found in Project Settings → API → Project API Keys → anon public
Credentials are saved in your browser. Only you see this setup screen.
